# **App Name**: Tyaari Coach

## Core Features:

- Interview Duration Selection: Presents users with a clear choice of interview durations (15, 30, or 45 minutes).
- Camera and Mic Access: Request permission to utilize the user's camera and microphone to facilitate the interview.
- Real-time Question Generation: AI-powered tool dynamically generates interview questions in real-time, customized for each interview.
- Conversational Interviewing: AI driven conversational interview tool guides the user through conversational style questions. API Key Required.
- Personal Dashboard: A results dashboard tracks user interview results and suggests study topics.

## Style Guidelines:

- Primary color: Saturated blue (#3F51B5), to inspire confidence. It must contrast against the background.
- Background color: Dark, slightly desaturated blue (#283593), establishing a professional, clean atmosphere.
- Accent color: Soft violet (#7C4DFF), intended for interactive elements and highlights.
- Maintain clean and modern typography to ensure readability.
- Utilize icons from the Material UI icon library, aligning with the platform's aesthetic.
- Employ a fluid and responsive layout to support access on diverse screen sizes.
